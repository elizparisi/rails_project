# README

BeFound

BeFound is an app built on Rails. Users can create an account with BeFound to make and view recommendations, and comment on other users recommendations.

Features

    Login with Google

Technologies

Project created with:

    Rails 6.0.2
    Bootstrap-Sass
    Sqlite3 1.4
